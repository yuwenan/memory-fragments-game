# 记忆碎片 · 故事圣经 v1（已锁）

> 唯一真相来源。做任何房间 / 闪回 / 谜题 / 最后一扇门，都以此为准。
> 核心铁律：**反转藏在玩家早就看过的细节里；"收集"本身就是陷阱。**

## 一句话内核
你以为在拼回自己的记忆，其实你正在被一笔笔抹去——每收集一块碎片，系统就把它录进你的「镜像」、再从你脑里删掉。进度条不是恢复，是删除。

## 设定
- **觉醒室**：不是牢房，是**提取舱**。游戏从这里开始。
- **镜像计划**：给每个"志愿者"做一份数字镜像（意识备份）。镜像出舱后将顶替你、过你本该过的人生。
- **墙上 FACILITY RULES = 整个真相的明示**（玩家会当成监狱规矩）：
  - Silence 别出声 · Obey 配合 · **Record 把你的记忆录进镜像** · **Forget 然后从你脑里删掉**
- **7931**：你的志愿者编号。你曾在清醒间隙借台灯最后的光把它刻进石墙，怕自己忘了"我是谁"，留个证据。
  - 谜面"从最亮的光读进黑暗"双关：从此刻的光，读回正在被吞掉的过去。
- **柜中"日期是明天"的照片**：镜像出舱后明天要去的地方——是**它的**生活，不是你的。

## 核心循环（机制即叙事）
1. 走廊选一扇门 → 进一个独立房间（独立画风/谜题/可能有怪物，会死亡）
2. 解谜拿到该房间的**记忆碎片** → 触发**记忆闪回**（揭示一段过去）
3. 玩家以为"找回一段自己"，实则该段被录进镜像、从你脑中删除
4. 顺序锁：通关当前门才解锁下一扇 → 全部碎片汇向**最后一扇门**

## 房间 = 不同记忆，画风就是线索（不是为花样而花样）
- **可爱房间**：别人的、你向往却从没拥有的童年（你童年是糟的，所以渴望它）。柔和绘本风。
- **像素房间**：你打过的游戏 / 逃进去的那块屏幕。像素风 = 你曾经的避难所。
- **恐怖房间**：你真正的创伤——父母、老师。最写实最压抑。
- （可继续扩：每扇门一段记忆，风格服务于那段记忆的情绪）

## 最后一扇门
你以为门后是出口/真相。门开了——对面站着"你"，穿你的脸，刚录满你全部记忆，平静地走出去过你的人生。
你变回觉醒室里被掏空、什么都不记得的空壳。灯灭。
（绕开时间循环：不是你重复同一天，是机构一个接一个处理"志愿者"，7931 之后还有 7932——流水线的恐怖。）

## 回味点清单（细思极恐的"原来如此"）
- 墙上 Record / Forget 开头就写明结局，你当成了规矩
- "日期是明天"的照片 = 镜像的未来，不是你的
- 你玩得越投入、收集越多，角色丢得越彻底（机制陷阱）
- 最向往的可爱童年，通关后归了镜像，你连"想要"的资格都没了

## 碎片文案（逐房间写，先种第一块）
- **碎片① 觉醒室（铜钥匙同时获得）**：第一道裂缝。不点破，只让人"咯噔"一下——
  闪回：刺眼的灯、皮带勒住手腕、一个平静的声音说"记录完成 73%……继续"。然后黑。
  你醒在椅子上，墙上是你自己刻的 7931。你不记得是什么时候刻的。
